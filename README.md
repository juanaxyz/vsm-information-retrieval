# Python TF-IDF SVM

## Instalasi

### Prasyarat
- Python 3.7 atau lebih tinggi
- pip (Python package manager)

### Langkah Instalasi

1. Clone repository
```bash
git clone <repository-url>
cd python-tf-idf-svm
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

Atau install manual:
```bash
pip install scikit-learn numpy pandas
```
